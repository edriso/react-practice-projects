# React.js Projects

A collection of React projects built while learning fundamental concepts through John Smilga's React course.

## Projects Overview

### 00. Backroads
**Landing Page for Travel Company**  
Converting a static HTML/CSS website to React components, practicing component composition and props.

### 01. Birthday Buddy
**Birthday List Application**  
- useState hook
- Rendering lists with map()
- Props and component structure
- Conditional rendering

### 02. Tours
**Tours Listing with API**  
- useState and useEffect hooks
- Fetching data from API
- Event handling
- Conditional rendering and loading states

### 03. Reviews
**Review Carousel**  
- State management
- Event handlers
- Dynamic content rendering
- Navigation logic (next, previous, random)

### 04. Accordion
**FAQ Accordion Component**  
- State management
- Toggle functionality
- Conditional rendering
- Controlled components

### 05. Menu
**Restaurant Menu with Filtering**  
- Component composition
- Filtering data by category

- Dynamic button generation

### 06. Tabs
**Job Experience Tabs**  
- Fetching data from external API
- useEffect for side effects
- Managing multiple state variables
- UUID library for unique IDs

### 07. Slider
**Image Carousel/Slider**  
- Building custom carousel

## Getting Started

Each project has its own dependencies. To run any project:

```bash
cd [project-folder]
npm install
npm run dev  # or npm start for older projects
```

## Technologies Used

- React.js
- Vite (build tool for most projects)
- Various npm packages (react-icons, nanoid, values.js, react-toastify, etc.)
- Redux Toolkit
- Context API
